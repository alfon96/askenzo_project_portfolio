AskEnzo APIs

Bridging Apps with Data in Brindisi

  • The AskEnzo APIs serve as the vital communication link between our app and webapp and the SQL relational databases hosted on Amazon RDS.
  • These APIs meticulously handle user information, experiences, discoveries, likes, and also interact with Amazon S3 for efficient image hosting.
  • Designed to cater to the needs of tourists in Brindisi and local accommodation providers like hotels and B&Bs, AskEnzo APIs facilitate these users to discover and offer unique experiences in the city of Brindisi.

FastAPI

Embracing Modernity with FastAPI

  • In our pursuit of a contemporary and robust framework for Python, we embraced FastAPI for its simplicity and efficiency.
  • This choice enabled us to develop the necessary functionalities for AskEnzo with remarkable ease, ensuring a seamless and dynamic user experience.

AWS ECS

Scalable Solutions with AWS ECS

  • To ensure that our APIs can scale in response to fluctuating user demands, we've integrated AWS ECS into our infrastructure.
  • The APIs are developed and maintained within Azure Git repositories, complete with Dockerfiles.
  • Our project also includes a YAML file that defines an automated pipeline.
  • This pipeline adeptly handles the code, packages it into Docker, and deploys it to AWS ECS, streamlining our development and release processes.

Data Management

Reliable Data Management with Amazon RDS

  • Amazon RDS is our chosen solution for storing all application data, except for images.
  • his approach guarantees a reliable, secure, and organized data management system, ensuring data integrity and accessibility for the AskEnzo app.

Stateless Architecture

Securing Access with JWT and bcrypt

  • Our goal of achieving a stateless architecture led us to implement JWT tokens for secure API access.
  • Additionally, we prioritize user security by encrypting all user passwords using bcrypt, ensuring both safety and privacy within the AskEnzo ecosystem.
